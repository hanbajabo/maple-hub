# 구글 애드센스 심사 대응 전략

> **작성일:** 2025-11-27  
> **목적:** maple.ai.kr 애드센스 승인을 위한 전략 문서

---

## 📋 현재 상태 진단

### ✅ 강점
- **기능:** 넥슨 오픈 API 활용한 완성도 높은 캐릭터 분석 도구
- **디자인:** 프리미엄급 UI/UX
- **기술:** Next.js SSR, TypeScript, 최신 웹 기술 스택
- **법적 준비:** 개인정보처리방침 완비

### ❌ 약점 (애드센스 관점)
- **텍스트 콘텐츠 부족:** 구글 봇이 읽을 정적 텍스트가 부족
- **유틸리티 사이트 한계:** 검색창 + 결과만 있는 구조
- **낮은 체류 시간:** 검색 → 결과 확인 → 이탈 패턴
- **스크랩 콘텐츠 오해 가능성:** 메이플스토리 소식 섹션

## 🎯 해결 전략

### 1단계: 블로그/가이드 섹션 신설 (최우선)

#### 구조 설계
```
/guide
├─ /hexa-stats-optimization      # 헥사 스탯 최적화 가이드
├─ /boss-equipment-progression  # 보스 장비 성장 가이드
├─ /combat-power-tier-system    # 전투력 티어 시스템 설명
├─ /starforce-efficiency        # 스타포스 효율 계산
├─ /union-board-setup            # 유니온 배치 가이드
├─ /link-skill-priority          # 링크스킬 우선순위
├─ /ability-reroll-strategy      # 어빌리티 재설정 전략
├─ /potential-cube-guide         # 잠재능력 큐브 가이드
├─ /symbol-force-farming         # 심볼/포스 파밍 루트
└─ /maple-ai-introduction        # 메이플 AI 소개 및 사용법
```

#### 각 가이드 포스트 작성 기준
- **분량:** 최소 1,000자 이상
- **구성:** 제목 → 서론 → 본론(단계별) → 결론 → FAQ
- **톤앤매너:** "단풍이 AI"의 친근한 목소리
- **이미지:** 스크린샷, 다이어그램 포함
- **데이터 기반:** 실제 넥슨 API 데이터 활용

#### 콘텐츠 예시

**제목:** "헥사 스탯 효율적으로 분배하는 수학적 원리"

**구성:**
1. **서론:** 헥사 스탯이 왜 중요한가?
2. **본론:**
   - 스탯별 효율 비교 (공격력 vs 크리티컬 데미지)
   - 직업별 추천 분배 비율
   - 단계별 성장 로드맵
3. **결론:** 메이플 AI를 활용한 자동 분석 권장
4. **FAQ:** 자주 묻는 질문
