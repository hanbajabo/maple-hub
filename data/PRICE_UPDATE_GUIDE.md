# 시세 데이터 업데이트 가이드

## ⚠️ AI에게 줄 프롬프트 (복붙용)

아래 프롬프트를 그대로 복사해서 AI에게 붙여넣으세요:

---

```
`data/item-price-trends-raw.md` 파일 끝에 아래 시세 데이터를 추가해줘.

**규칙 (반드시 지킬 것):**
1. 파일 인코딩은 반드시 UTF-8로 저장할 것
2. 추가할 위치는 파일의 맨 끝 (기존 내용 수정 없이 append)
3. 형식은 아래 템플릿을 정확히 따를 것
4. 에테르넬 평균가는 직접 계산할 것: (전사+마법사+궁수+도적+해적) / 5
5. 가격이 없는 항목은 줄 자체를 넣지 말 것 (- 로 표기 금지)
6. 익셉 - 훈장처럼 가격이 "-"인 항목은 생략할 것

**추가할 데이터:**
[여기에 표 붙여넣기]

**템플릿:**
### YYYY-MM-DD

#### 1. 챌린저스 서버 (챌섭)
- 아이템명: 가격
...

#### 2. 본 서버 (본섭) (단위: 억)
- 아이템명: 가격
...

#### 3. 에테르넬 장비 (평균가 단위: 억)
- 모자: X.XX (전X.XX/마X.XX/궁X.XX/도X.XX/해X.XX)
- 상의: X.XX (전X.XX/마X.XX/궁X.XX/도X.XX/해X.XX)
- 하의: X.XX (전X.XX/마X.XX/궁X.XX/도X.XX/해X.XX)
- 견장: X.XX (전X.XX/마X.XX/궁X.XX/도X.XX/해X.XX)
- 신발: XX.XX (전XX.X/마XX.X/궁XX.X/도XX.X/해XX)
- 장갑: XX.XX (전XX.X/마XX/궁XX.X/도XX.X/해XX.X)
- 망토: XX.XX (전XX/마XX.X/궁XX/도XX/해XX)

#### 4. 광휘의 보스 세트 (평균가 단위: 억)
- 근원의 속삭임: 가격
- 죽음의 맹세: 가격
- 불멸의 유산: 가격
- 황홀한 악몽: 가격
```

---

## 📋 아이템명 통일 규칙

파서가 인식하는 정확한 아이템 이름:

| 입력할 이름 | 비고 |
|---|---|
| `자석펫 7기(평균)` | 괄호 형식 주의 |
| `불안정한 시간의 파편` | 풀네임 |
| `블랙하트` | 블랙하트(챌카) X |
| `에리온의 조각` | |
| `익셉 - 벨트` | 띄어쓰기 정확히 |
| `익셉 - 얼장` | |
| `익셉 - 눈장` | |
| `언더컨트롤` | |

## 🚨 자주 발생하는 오류

| 문제 | 원인 | 해결 |
|---|---|---|
| 기타템/자석펫 누락 | 인코딩 깨짐 (EUC-KR) | 반드시 UTF-8로 저장 지시 |
| 섹션 전체 누락 | `####` 헤딩 인식 실패 | 인코딩 문제와 동일 원인 |
| 에테르넬 직업별 파싱 실패 | 평균값 형식 오류 | `(전X/마X/궁X/도X/해X)` 형식 준수 |
